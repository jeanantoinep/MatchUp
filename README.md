# FRONTEND

(prerequisite : yarn, expo)

```
cd Front
cp .env.template .env
// write your API_URL (http://<your_ip_address>:3000)
yarn
yarn start
//connect to the app using Expo Go app or Android Studio/Xcode
```

# BACKEND

```
cd Api
cp .env.template .env
write your mongodb DATABASE_URL
npm i
npm start
```
